# Skills (source folder)

The Skills section renders at https://product-on-purpose.github.io/pm-skills/skills/.

**This README is the GitHub directory landing page.** The canonical user-facing overview lives in [`index.md`](index.md), which is what renders on the docs site at `/skills/`. Per-skill pages under this directory (`discover/`, `define/`, `develop/`, `deliver/`, `measure/`, `iterate/`, `foundation/`, `utility/`) are auto-generated by `scripts/generate-skill-pages.py` from the source files in `/skills/<name>/SKILL.md`. To edit a skill's content, update the source SKILL.md and re-run the generator.
