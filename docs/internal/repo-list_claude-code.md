### Submission Notes:

- Ensure each submission references the agentskills.io specification compliance
- Highlight the 24 PM skills across 6 Triple Diamond phases
- Emphasize Apache 2.0 licensing
- Include link to live repository and documentation
- For PM-focused repos, emphasize the product management workflow benefits
- For agent skills repos, emphasize spec compliance and multi-use cases
