# Releases (source folder)

The Releases section renders at https://product-on-purpose.github.io/pm-skills/releases/.

**This README is the GitHub directory landing page.** The canonical user-facing overview lives in [`index.md`](index.md), which is what renders on the docs site at `/releases/`. Per-release notes (`Release_vX.Y.Z.md`) are hand-authored at release time and follow the format established in v2.5.0 onward. The root [`CHANGELOG.md`](https://github.com/product-on-purpose/pm-skills/blob/main/CHANGELOG.md) is the per-version Keep-a-Changelog source-of-truth; release notes here are the user-facing narrative complement.
